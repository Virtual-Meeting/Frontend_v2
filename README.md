# 🖥 VMO [ Vision + Motion / Emoji ] 

**비언어적 표현**을 강화한 **실시간 커뮤니케이션 플랫폼**


<br/>



## 📝 프로젝트 개요
VMO는 언어뿐 아니라 **표정, 제스처, 반응**을 통해 **비언어적 소통을 강화**한 실시간 커뮤니케이션 플랫폼입니다.

- 이모지를 활용한 반응을 통해 상대에게 감정과 반응을 더 **직관적**이고, 풍부하게 표현 
- 줌, 구글 Meet 등 기존 화상회의의 한계를 넘어 몰입감 있고 자연스러운 소통을 목표

<br/>


| 🔎 해결하고자 하는 문제  | 💡 핵심 솔루션 |
| -- | -- |
| 🤐 감정, 방응 등의 **비언어적 표현 부재** <br/> 🤷 발화자의 **의도 전달**이 불명확 <br/> ❔ 청자의 **참여 여부** 파악 어려움 | 동적인 이모지 + 화상 커뮤니케이션 <br/>→ 표현력 향상 + 오해 최소화 + 몰입도 상승

<br/>

### 🎉 주요 기능

- **이모지 애니메이션**을 통해 **실시간 감정 표현/반응** 전달 (ex: 😍, ✋ 등)
- **회의 녹화 기능** (방장: 전체 회의 녹화, 참가자: 방장 승인 하에 녹화 가능)
- 발표자/참가자 간 **화면 공유**
- 실시간 화상 회의 중 **텍스트로 커뮤니케이션** 가능
